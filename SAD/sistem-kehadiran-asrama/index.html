Panduan Import Fail Excel

Format Fail Excel yang Diperlukan

Fail Excel anda mesti mengandungi lajur-lajur berikut:

Nama Lajur Keterangan Contoh Wajib
Name Nama penuh pelajar Ahmad bin Rahman Ya
ID ID pelajar yang unik 20240001 Ya
Class Nama dorm/blok Dorm A Tidak
Phone Nombor telefon 012-3456789 Tidak
Email Alamat email ahmad@sekolah.edu.my Tidak

Langkah-langkah Import

  1. Sediakan fail Excel:
    • Gunakan format .xlsx atau .xls
    • Pastikan baris pertama mengandungi nama lajur
    • Jangan tinggalkan baris kosong di antara data
  2. Klik butang "Import Excel" di tab Pelajar
  3. Pilih fail Excel dari komputer anda
  4. Tunggu proses import selesai
  5. Semak data yang telah diimport

Muat Turun Template

Untuk memudahkan anda, muat turun template Excel yang telah disediakan:

Perkara Penting

  • Pastikan ID pelajar adalah unik (tidak berulang)
  • Jika ID sudah wujud, data akan dikemaskini
  • Nama lajur mesti tepat seperti yang dinyatakan
  • Fail maksimum: 5MB
  • Maksimum 1000 pelajar setiap import

Panduan Jana Laporan Excel

Jenis Laporan yang Boleh Dijana

Laporan Harian

Kehadiran semua pelajar untuk hari tertentu

Laporan Mingguan

Ringkasan kehadiran untuk seminggu

Laporan Bulanan

Statistik kehadiran untuk sebulan

Laporan Mengikut Dorm

Kehadiran dikumpulkan mengikut dorm

Cara Jana Laporan Excel

  1. Pergi ke tab "Laporan"
  2. Pilih tarikh mula dan tamat:
    • Untuk laporan harian: pilih tarikh yang sama
    • Untuk laporan tempoh: pilih tarikh berbeza
  3. Klik "Jana Laporan" untuk melihat data
  4. Klik "Export ke Excel" untuk muat turun
  5. Fail akan dimuat turun secara automatik

Kandungan Laporan Excel

Laporan Excel akan mengandungi maklumat berikut:

Lajur Keterangan
ID Pelajar Nombor ID unik pelajar
Nama Nama penuh pelajar
Dorm Nama dorm/blok
Tarikh Tarikh kehadiran
Masa Masa kehadiran direkod
Status Hadir Asrama/Prep/Keluar/Lewat/Pulang Bermalam

Tips Berguna

  • Export laporan secara berkala untuk sandaran
  • Gunakan filter Excel untuk analisis lanjut
  • Buat pivot table untuk statistik yang lebih detail
  • Simpan laporan dengan nama yang mudah difahami
